# Linked List

A linked list is a chain of nodes where each node holds a value and a pointer to the next node.

## Methods
- **addNode** -- adds a new node to the end of the list
- **removeNode** -- removes a node by value
- **findMin** -- returns the smallest value in the list
- **findMax** -- returns the largest value in the list
- **search** -- returns true if a value exists in the list
- **reverse** -- reverses the order of the list
